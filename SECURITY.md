# Security Policy

## Supported Versions

Viene supportato il branch `main` (ultima versione pubblicata).

## Reporting a Vulnerability

Se trovi una vulnerabilita, non aprire una issue pubblica.

Usa una segnalazione privata:

- GitHub Security Advisory (preferito)
- oppure contatta direttamente i maintainer del repository

Includi:

- descrizione del problema
- passi per riprodurre
- impatto atteso/reale
- eventuale PoC

## Response SLA (obiettivo)

- presa in carico: entro 72 ore
- valutazione iniziale: entro 7 giorni
- fix o mitigazione: in base alla severita
